# Nova Launcher

A modern, feature-rich Minecraft launcher with full mod support.

![Nova Launcher](icon.png)

## Features

- 🎮 **Multiple Instances** - Create separate instances for different modpacks
- 🔧 **Mod Management** - Install, enable/disable, and manage mods easily
- 🌐 **Mod Browser** - Search and install mods from Modrinth and CurseForge
- 📦 **Modpack Support** - Export and import modpacks
- 🎨 **Multiple Loaders** - Supports Vanilla, Fabric, Forge, and Quilt
- 🔐 **Microsoft Authentication** - Secure login with Microsoft accounts
- 💾 **Automatic Java Management** - Downloads the correct Java version automatically
- 🖥️ **System Tray Support** - Hide to system tray with custom dialog

## Installation

### Portable Version (Recommended)
1. Download `Nova-Launcher.zip`
2. Extract to any folder
3. Run `Install-Shortcut.bat` (optional, adds to Start Menu)
4. Run `Nova Launcher.exe`

### First Time Setup
1. Click "Add Microsoft Account" to log in
2. Create a new instance with your preferred Minecraft version
3. Choose your mod loader (Vanilla, Fabric, Forge, or Quilt)
4. Click "Play" to launch!

## System Requirements

- Windows 10 or later (64-bit)
- 4GB RAM minimum (8GB+ recommended for modded)
- Minecraft Java Edition account
- Internet connection for first-time setup

## Controls

| Action | How |
|--------|-----|
| Close Launcher | Click X → Choose "Hide to System Tray" or "Quit" |
| Restore from Tray | Double-click tray icon or right-click → "Show Nova Launcher" |
| Add Mods | Drag and drop `.jar` files into the Mods page |

## Troubleshooting

### "Windows protected your PC" warning
This is normal for unsigned apps. Click **More info** → **Run anyway**.

### Smart App Control blocked the file
1. Right-click the file → Properties
2. Check "Unblock" (if available)
3. Click Apply → OK
4. Run again

### Launcher won't start
- Make sure you have [Visual C++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe) installed
- Try running as Administrator
